Linux

Linux® es un sistema operativo (SO) open source que creó Linus Torvalds en 1991. Actualmente, tiene una base de usuarios masiva y lo utilizan las 500 supercomputadoras más potentes del mundo. Los usuarios lo eligen por su versatilidad y funciones de seguridad, entre otras razones. Al kernel de Linux lo mantiene una comunidad mundial de aficionados open source y tiene cientos de distribuciones únicas. Linux se diseñó para que se parezca a UNIX, pero ha evolucionado para ejecutarse en una gran variedad de sistemas de hardware, desde teléfonos hasta supercomputadoras. Todos los sistemas operativos basados en Linux incluyen el kernel, que gestiona los recursos de hardware, y un conjunto de paquetes de software que conforman el resto del SO. Las empresas también pueden elegir ejecutar el sistema en un servidor de Linux.

Este sistema operativo incluye algunos elementos esenciales comunes, como las herramientas GNU,las cuales permiten que el usuario gestione los recursos que brinda el kernel, instale sistemas de software adicionales y configure el rendimiento y la seguridad, entre otras tareas. Todas esas herramientas juntas forman un sistema operativo funcional. Debido a que Linux es un SO open source, las combinaciones del software pueden variar entre distribuciones.
